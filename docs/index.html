Why Use Video Enhancer?

🚀

4K Upscaling

Intelligent upscaling using Lanczos algorithm with sharpening enhancement. Transform 720p/1080p footage into crisp 4K.

⚡

FPS Boosting

Motion-compensated frame interpolation creates smooth high-frame-rate video. ​Go from 30fps to 60fps or even 120fps.

📦

Batch Processing

Process entire folders of videos at once. Set it and walk away — perfect for large libraries.

🎯

Audio Preserved

Original audio is preserved untouched through the enhancement pipeline. No quality loss on sound.

Quick Usage

python enhance.py video.mp4 --to-4k
Upscale any video to 4K resolution
python enhance.py video.mp4 --to-4k --fps 60
4K upscale + smooth 60fps
python enhance.py video.mp4 --to-4k --fps 120
4K upscale + ultra-smooth 120fps
python enhance.py ./videos/ -o ./enhanced/ --to-4k
Batch process entire folders
python enhance.py video.mp4 --info
Preview video info before processing
python enhance.py video.mp4 --to-4k --hwaccel
Use NVIDIA GPU acceleration

Get Started in Seconds

bash 
# Clone the repo
git clone https://github.com/ikechukwuotis6928579/video-quality-enhancer.git
cd video-quality-enhancer

# Install dependencies
pip install -r requirements.txt

# Install FFmpeg (if not installed)
sudo apt install ffmpeg -y

# Enhance your first video
python enhance.py your-video.mp4 --to-4k --fps 60
